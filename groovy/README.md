# mcandre/groovy

Introductory programs in Groovy

# EXAMPLE

```
$ cd hello/
$ mvn package
$ mvn exec:java -Dexec.mainClass=us.yellosoft.hello.groovy.Hello
Hello World!
```

# REQUIREMENTS

* [Maven](http://maven.apache.org/) 3+
* [JDK](http://www.oracle.com/technetwork/java/javase/downloads/index.html) 1.7+
